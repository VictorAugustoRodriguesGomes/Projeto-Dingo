Sobre o Projeto

Olá! Meu nome é Victor Augusto e sou responsável por desenvolver esse projeto nomeado com Projeto Dingo (utilizei esse nome genérico para organização pessoal).

O objetivo de demonstrar as minhas habilidades como desenvolvedor.

Desenvolvi esse projeto com a finalidade de reproduzir diferentes músicas, o projeto conta com 20 música em 4 gêneros diferentes.

O projeto conta com um controlador, onde é possível aumentar e diminuir a velocidade e volume da música, também é possível alterar as músicas pressionando as telhas de atalhos.

Por mais que esse projeto possua algumas funções interessantes, acabei não gostando do resultado e não continuei o desenvolvimento do projeto, mas, ele serviu como inspiração e aprendizado para outros projetos futuros.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript.
